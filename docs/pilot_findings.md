# Pilot Findings

## Scope

The pilot analyzes monthly U.S. imports of memory integrated circuits under HS code 854232 during 2024.

The purpose was to validate the complete analytical pipeline:

UN Comtrade  
→ Snowflake raw layer  
→ Staging transformation  
→ Analytics view  
→ Executive insight

## Initial Finding

The three largest reported supplier markets accounted for approximately 70.28% of U.S. import value for memory integrated circuits in 2024.

The leading reported supplier markets were:

| Rank | Supplier market | Import value | Supplier share |
|---:|---|---:|---:|
| 1 | Other Asia, nes | $652.9M | 34.70% |
| 2 | Republic of Korea | $433.6M | 23.05% |
| 3 | Japan | $235.8M | 12.53% |
| 4 | Malaysia | $210.6M | 11.19% |
| 5 | China | $94.1M | 5.00% |

## Executive Interpretation

The result indicates that U.S. memory-integrated-circuit imports were concentrated among a relatively small group of supplier markets during the pilot period.

From an executive perspective, this raises three questions:

1. Is the concentration persistent across multiple years?
2. Does the same pattern appear across other semiconductor categories?
3. Which supplier markets could provide realistic diversification capacity?

These questions will be addressed as the project expands.

## Limitations

This is a pilot result and should not be interpreted as evidence of company-specific semiconductor dependency.

The analysis:

- Covers only the United States
- Covers only 2024
- Covers only memory integrated circuits under HS 854232
- Uses trade value rather than physical production capacity
- Does not identify purchasing companies or final products
- Retains the source label `Other Asia, nes`, which requires careful treatment in later country-level analysis

The result describes reported international trade concentration. It does not by itself measure operational risk, supplier capacity, or the ability of alternative markets to replace disrupted supply.
